Edge Detection

Edge detection can be done in different ways, all with linear algebra at their core. As you know, images can be represented with matrices. These matrices can then be manipulated to do cool things such as edge detection. First, we will introduce a new matrix operation called kernel convolution. Never heard of that? Think it sounds like corn? Are you getting hungry for popcorn? Well, there's no popcorn, but it does involve taking a matrix called a kernel and traversing another matrix, taking same-sized kernels out of that. The kernels of that matrix are centered around the specific element that you're on when traversing the matrix. The elements of the two kernels are multiplied together. The elements of the resulting kernel are then averaged. This average is then used to replace the specific element of the matrix (the one that isn't the kernel). By replace, we actually mean that you should put it in a separate same-sized matrix so you can preserve the values of the original matrix.
